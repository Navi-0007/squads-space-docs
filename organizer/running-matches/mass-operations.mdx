title: "Mass Operations"

Mass Operations lets you manage matches across all groups or pairings in a stage from a single page — instead of navigating to each group individually. This is essential for large tournaments with many groups.

## Mass Create

Create matches for multiple groups or pairings at once. The flow has four steps:

### 1. Select Groups or Pairings

Choose which groups (Elimination) or pairings (Round Robin) you want to create matches for. You can select all of them or pick specific ones.

### 2. Match Counts

Set how many matches to create for each selected group or pairing. Each one can have a different count — you don't have to use the same number for every group.

### 3. Scheduling

Choose how match times are generated:

- **Sequential** — matches are spaced by an interval from a start time. You set the interval (minimum 20 minutes) and choose the scope:
  - **Per group** — each group runs its own timeline. You can optionally set a custom start time for each group.
  - **Global** — one continuous timeline across all selected groups. Every next match starts after the interval, regardless of which group it belongs to.
- **Same Time** — all matches start at exactly the same time. Use this when all groups play simultaneously.
- **Skip for now** — create matches without start times. You can schedule them later from Mass Manage.

For sequential scheduling, you can use interval presets (20, 30, 45, or 60 minutes) or enter a custom interval.

### 4. Maps

Choose the map for each match:

- **Same for all** — one map for every match across all selected groups
- **Custom per group** — select a different map for each match in each group. For groups with more than 2 matches, maps must be selected in order.

## Mass Manage

After matches are created, Mass Manage lets you bulk-edit schedules and match details across all groups from one page. This is where you go to schedule matches that were created with "Skip for now", or to update times across multiple groups at once.

## Mass Results

Enter scores for matches across all groups from a single page. Select which group and match to update, enter results, and move to the next — without navigating back and forth between group pages.

## When to Use Mass Ops

Mass Ops is designed for large stages with many groups. If your stage has 2-3 groups, managing them individually is fine. But when you're running 8, 12, or 16 groups in a qualifier stage, Mass Ops saves significant time:

- Create all matches at once with proper scheduling across groups
- Set maps in bulk
- Enter results across groups without constant page navigation
- Keep scheduling consistent across the entire stage
