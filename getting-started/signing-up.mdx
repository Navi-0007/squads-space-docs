SquadsSpace uses a single auth page for both new and returning users.

## Creating an Account

1. Go to [platform.squads-space.com/auth](https://platform.squads-space.com/auth)
2. Enter your email address
3. If you're new, you'll be prompted to create an account — set a password and fill in your display name and username
4. Verify your email with the code sent to your inbox
5. You're in — the dashboard loads automatically

## Signing In

1. Go to the same auth page
2. Enter your email address
3. Enter your password
4. If you have two-factor authentication enabled, enter the code from your authenticator app

## Passkeys

SquadsSpace supports passkeys as an alternative sign-in method. If your device supports it, you can set up a passkey from Settings → Security for faster, passwordless sign-in.

## Forgot Password

If you've forgotten your password, use the "Forgot password" link on the sign-in screen. You'll receive a reset link by email.

## What's Next

After signing in, you'll land on your Player Dashboard. From there you can browse tournaments, manage your squad, or become an organizer.
