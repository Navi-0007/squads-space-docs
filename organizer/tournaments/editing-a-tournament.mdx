---
title: "Editing a Tournament"
description: "Update tournament details after creation."
---

You can edit your tournament after creating it. From the Tournament Dashboard, go to Edit to open the editing wizard.

## What You Can Edit

The edit page uses the same wizard layout as creation. You can update:

- Name, description, and poster
- Visibility and registration mode
- Schedule dates (registration open/close, tournament start/close)
- Requirements
- Communication links (WhatsApp, Discord)
- Point table configuration

## Draft Tournaments

If your tournament is still in Draft status, you'll see a banner with the option to publish directly from the edit page. Draft tournaments aren't visible to players until published.

## Restrictions

Some fields may be restricted based on the tournament's current status. For example, you can't change the game mode or format after registrations have started, since it would affect existing team compositions.
